# HertzTest

Comprehensive (soon) integration tests for the [HertzScript](https://github.com/Floofies/HertzScript) compiler!

## Usage

```shell
git clone https://github.com/trgwii/HertzTest.git
cd HertzTest
npm i
```

Linux:
```bash
./scripts/compile.sh
./scripts/run.sh
```

Windows:
```cmd
scripts\compile
scripts\run
```
